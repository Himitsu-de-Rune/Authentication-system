# Собственная система аутентификации и авторизации

## Структура БД

### Таблица users
| Поле          | Тип      | Назначение        |
|---------------|----------|-------------------|
| id            | int PK   | ID пользователя   |
| first_name    | text     | Имя               |
| last_name     | text     | Фамилия           |
| email         | text     | Уникальный логин  |
| password_hash | text     | Хеш пароля        |
| is_active     | bool     | Мягкое удаление   |
| role_id       | int FK   | Роль пользователя |

### Таблица roles
| id | name          |
|----|---------------|
| PK | Название роли |

### Таблица permissions
| id | role_id | resource | action |
|----|---------|----------|--------|

Пример:
| role_id | resource | action |
|---------|----------|--------|
| 1       | items    | read   |
| 1       | items    | write  |
| 2       | items    | read   |

### Таблица sessions
| id | user_id | token  |
|----|---------|--------|
Хранит активные сессии.
